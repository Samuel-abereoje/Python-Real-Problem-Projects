Note Manager (CLI Application)

A simple and efficient Command-Line Note Management System built with Python.
This application allows users to create, view, delete, and search notes, with data persisted locally using JSON.

Features
Add notes with timestamps
View all saved notes
Delete notes by ID
Search notes by keyword
Persistent storage using JSON
Clean and minimal CLI interface


The system follows a simple data lifecycle:
Load existing notes from notes.json
Perform user action (add, delete, search, view)
Save updated notes back to file

This project demonstrates:
File I/O operations
Data serialization (JSON)
CLI application design
Basic CRUD operations
Modular programming in Python
