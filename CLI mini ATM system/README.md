Command-Line mini ATM system(Python)  

Overview

This project is a command-line mini banking application built with Python. It simulates core banking operations such as balance inquiry, withdrawals, deposits, and transaction tracking.

The goal of this project is to demonstrate foundational programming concepts, clean code structure, and practical system design thinking.

Features

Check account balance
Withdraw funds with validation
Deposit funds securely
Transaction history tracking
Input validation and error handling
Continuous user interaction via CLI

Key Learning Outcomes

Structured programming using functions
Writing maintainable and readable code
Handling invalid user input gracefully
Managing application state (balance & transactions)
Building interactive CLI applications

Future Improvements

Add PIN authentication system
Store data using SQLite database
Support multiple user accounts
Export transactions to CSV for analysis
Build a web interface (Flask/Django)
